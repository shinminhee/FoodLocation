

# FoodLocation-ios

붕세권 App을 Clone하여 만든 어플리케이션 입니다.

```swift
붕세권 App을 Clone 하여 개발함으로써, 네이버 지도를 사용하였고, 카카오 우편 주소 서비스를 이용하여 위치 서비스도 가능하게 만들었습니다. 또한 구글 로그인이 가능하게 구현하여 사용자의 닉네임으로 가게를 직접 등록할 수 있도록 하는 것을 목표로 기획하였습니다.
```
